## FRAMEWORKS QUE TRAZ PARA A VAIANUNCIO

### Para Operação de Tráfego
**Os 4 Números Obrigatórios** — antes de gastar $1, conhecer:
- **EPC** (Earnings Per Click): quanto ganha por clique → define o bid máximo
- **CPA** (Cost Per Acquisition): quanto custa adquirir → compara com LTV
- **AOV** (Average Order Value): ticket médio → determina lucratividade
- **DPL** (Dollars Per Lead): valor por lead → define quanto pagar para gerar leads

> Aplicação: A VaiAnuncio conhece os 4 números dos seus próprios clientes? Se não, não pode escalar com segurança.

**Test-Learn-Earn Protocol** — metodologia de escala em 3 fases:
1. Teste em escala mínima (valida premissas)
2. Aprende com os dados (não com opiniões)
3. Escala o que foi provado

### Para Modelos de Plataforma
**Knowledge Network (SVOD)** — plataforma de assinatura de conteúdo com:
- Nível gratuito (Product-Led Growth)
- Assinatura ~$19.99/mês (reduz churn drasticamente)
- Certificações e upsells para elevar ARPU acima do $60 CPA threshold

> Aplicação: A VaiAnuncio pode criar uma rede de conhecimento de tráfego para clientes — diferenciando-se como autoridade além da gestão operacional.

**Expert Partnership Acquisition** — abordagem sistemática para parcerias:
> Monitorar lançamentos de livros/cursos e propor: promoção em troca de conteúdo para a plataforma.
> YouTubers são alvos ideais — anos de conteúdo prontos para empacotar.

### Para Geração de Leads da Agência
**$60 CPA Threshold** — benchmark de mercado atual para aquisição via paid ads.
> Se a VaiAnuncio não conhece o custo de aquisição de um novo cliente, está operando no escuro.

**Churn Rate < 10% como pré-requisito de escala** — crescer com 30% de churn exige 40% de crescimento só para ficar no mesmo lugar.
