## 🗺️ FLUXOS RECOMENDADOS

### Ciclo Completo de Fechamento
```
/prospectar (gera lista + scripts por nicho)
  → n8n dispara automaticamente (após /n8n-setup configurado)
    → Claude qualifica respostas automaticamente
      → Fran recebe alerta: "Lead qualificado"
        → /proposta (gerar proposta para a reunião)
          → /reuniao-vendas (script Octane + NEPQ)
            → /onboarding-cliente (assinou!)
              → Kauan + Rodrigo (campanhas)
                → /nurture (lead não fechou ainda)
```

### Ciclo de Conteúdo Orgânico
```
/conteudo ou /carrossel (criar post)
  → /hookify (refinar gancho)
    → /comfyui via Aria (visual)
      → agendar publicação
```

### Ciclo de Decisão Estratégica
```
/empresa (sessão completa)
  → /research-first (se precisar de dados extras)
    → /writing-plans (plano de execução)
      → /executing-plans (implementar)
        → /chronicler (registrar decisão)
```

### Ciclo de Nova Campanha para Cliente
```
/research-first (benchmark do nicho)
  → Kauan (/empresa Meta) ou Rodrigo (/empresa Google)
    → /landing-page ou /web-designer (criar LP)
      → /site (deploy no ar)
        → monitorar métricas (CPL, CPA, DPL)
```

### Onboarding de Novo Expert/Conteúdo
```
/gdrive-transcription-downloader (baixar e transcrever)
  → /jarvis (processar pipeline completo)
    → /treinar [agente] (alimentar o agente certo)
      → /chronicler (registrar aprendizado)
```
